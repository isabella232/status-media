### Installation

1. Clone the repo

```sh
git clone https://github.com/jinhojang6/status-podcast.git
```

2. Install dependencies

```sh
yarn install
```

3. Start up the development server

```sh
yarn run dev

#visit http://localhost:1234/
```

4. Build

```sh
yarn run build
```
